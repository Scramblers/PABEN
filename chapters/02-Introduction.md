# 2. Introduction

Any theory of intelligence must eventually answer a simple but demanding question: how does an agent continue?

An intelligent system is not merely a system that represents the world, solves isolated problems, or selects actions from a menu. It is a system that must remain situated, preserve access to what matters, notice when its current path is no longer sufficient, and reorganize its behavior when the world, the body, or the need-state changes. In this sense, intelligence is not only a matter of computation. It is a matter of continued agency.

This starting point is related to several traditions that have emphasized cognition as embodied, situated, action-oriented, and environmentally embedded, rather than as detached symbol manipulation or passive representation (Gibson, 1979; Varela, Thompson & Rosch, 1991; Clark & Chalmers, 1998). It also resonates with later attempts to treat perception, action, and learning as parts of one regulatory loop, as in predictive processing and active inference frameworks (Friston, 2010).

This creates a problem for models that treat perception, action, cognition, emotion, attention, memory, and need as separate domains. Each of these can be studied in isolation, but an agent cannot live them in isolation. Perception must matter for action. Action must matter for need. Need must matter for attention. Experience must shape expectation. Emotion must signal when expectation changes. Consciousness must occupy the place where the next unresolved consequence becomes most important.

The PABEN model begins from this requirement of integration. It asks what kind of minimal functional architecture would be needed for an agent to recognize contexts, release skills, form beliefs, learn from experience, respond to needs, and regulate attention through changes in expectancy. The ambition is not to describe every biological detail of the brain, but to identify a small set of functional relations that may be necessary for intelligent behavior to emerge as one coherent process.

At the center of the model is the idea that agency depends on maintaining viable pathways from the present context toward fulfillment. The agent does not simply predict the world; it evaluates whether its current beliefs and actions keep meaningful continuation open. Expectation is therefore not treated as a detached estimate of truth, but as the felt and functional stability of a possible path: can this context hold, can this skill be executed, can this experience lead onward, and does the path preserve access to value?

On this view, emotion is not an additional module added to cognition. Emotion is the system’s response to changes in the expected viability of its current or possible paths. This is broadly compatible with work that has challenged a strict separation between emotion, bodily regulation, valuation, and rational action (Damasio, 1994). Attention is not a spotlight detached from motivation. It is drawn toward the beliefs where unresolved changes in expectancy matter most. Consciousness is not assumed to be a general container for all processing, but a limited mode of occupation where the agent must stabilize, redirect, or reconstruct its next continuation. This limited-access view has some affinity with global workspace approaches, while differing in its specific emphasis on expectancy, need, and action-continuation (Baars, 1988; Dehaene & Naccache, 2001).

The model assumes that learning is organized backwards from lived consequence. An agent does not first build a neutral map of the world and then later attach value to it. It discovers contexts through episodes in which actions, needs, pain, relief, and fulfillment have already been lived. Beliefs are therefore not detached propositions about the world. They are action-beliefs: learned ways of getting back to contexts in which regulation, relief, avoidance, repair, or fulfillment can again become available. A belief says, in effect: from this kind of context, this kind of action can return the agent to a context where something matters can be handled.

The model is organized around five domains: Perception, Action, Beliefs, Experience, and Needs. These are not meant as isolated components, but as mutually dependent functional domains in a single cycle of agency. Needs open pressure. Experience opens possible transitions. Beliefs bind contexts and skills into usable action-claims. Perception stabilizes the recognized context. Action releases and tests the available skill. The result is evaluated through changes in expectancy, which in turn regulate emotion, attention, learning, and process switching.
The purpose of the model is to make this cycle explicit. If intelligence is understood as continued agency under changing conditions, then the central problem is not representation alone, action selection alone, emotion alone, or consciousness alone. The central problem is how an agent keeps a path open from where it is, through what it can do, toward what matters.
