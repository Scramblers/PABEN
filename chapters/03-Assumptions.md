# 3. Assumptions

The PABEN model rests on a small set of structural assumptions. These assumptions define how the model treats agency, perception, learning, emotion, attention, consciousness, and memory from the beginning. They are not added as later explanations. They are the commitments that make the model different from an architecture built around passive representation, abstract reasoning, error correction, or detached cognition.
The central claim is that an agent is a continuity-seeking system. It must keep a viable path open from its current context, through possible action, toward need-regulation, avoidance, relief, repair, or fulfillment. The following cornerstones describe what follows from that starting point.

### There Is No Error Function
The model does not assume a global error function that compares an internal representation with the external world. A belief is not corrected by an abstract error-calculator. A belief is lived, and the world either carries it, fails to carry it, or opens another route.
Learning is therefore not driven by error as such. It is driven by lived confirmation, non-confirmation, closure, interruption, and re-routing. When a belief is carried through an episode, the episode strengthens or weakens future expectancy. When a belief cannot be carried, the agent must stabilize locally, reflect upward, or abandon the route. There is no need for a separate error substance. The episode itself contains the consequence.

### The Agent Sees What It Can Do
Perception is not neutral representation. The agent does not first construct a world-picture and then later decide what can be done in it. It recognizes contexts as possible carriers of belief and action.
A context is therefore not merely “what is there.” It is a recognized possibility-field: a structure in which certain skills can be prepared, certain beliefs can be opened, and certain routes can become available or blocked. The agent sees surfaces, objects, agents, situations, and relations through their relevance for possible continuation. Perception confirms and disturbs the agent’s active belief-world.

### The Agent Learns Backwards from Lived Consequence
The model assumes that meaning is learned backwards from consequence. The agent does not begin with neutral contexts that later receive value. Contexts become meaningful because they have been lived in episodes involving pain, relief, avoidance, repair, regulation, or fulfillment.
A Need does not point directly to a context. Need-pressure opens Experience. Experience retrieves episode-history: contexts in which lived beliefs have previously carried felt need-consequences. Beliefs therefore become ways of getting back to contexts where something can again be handled. Learning starts from what mattered and works backward into the contexts, skills, and beliefs that made it reachable.

### Emotion Is a Control Signal
Emotion is not decoration added to cognition. Emotion is the felt deformation of expectancy, dX. It tells the agent that the viability of a belief-path has changed.
When X rises, the agent feels that continuation has become more open, safe, reachable, valuable, or fulfilled. When X falls, the agent feels that continuation has become more threatened, blocked, unstable, costly, or lost. Emotion directs attention, process-switching, learning pressure, and prioritization. It is the control signal that tells the system where continuity must be protected, pursued, repaired, or released.

### Skills Are Prepared in Parallel; the World Selects the Sequence
The agent does not consciously compute every action step. A higher-level intended belief prepares a skill-structure, and this preparation can cascade downward into lower-level beliefs and skills. Many action possibilities can be primed at once.
Execution unfolds through the context-skill zip. Prepared skills wait for contexts that can release them, and recognized contexts expose the next possible skill-release. The world therefore participates in sequencing. The agent prepares a structured field of possible action, but the changing Context-surface determines which parts can actually unfold. Skilled action is not a pre-written command chain. It is a live coupling between prepared skill and recognized context.

### Experience Is the Statistical Ground of Expectancy
Experience is not storage of pictures. It is a graph of lived episodes. Each episode binds a starting context, a lived belief, a felt need-consequence, and a resulting context:

$$Ep_j=(C_{from},B_{lived},N_{felt},C_{to} )$$

Across many episodes, Experience contains distributions. These distributions allow the agent to read expectancy. Trustability can be read from how reliably a context carries a belief without dissipating. Executability can be read from how reliably a belief or skill remains coherent during execution. Reachability can be read from whether episode-fields provide routes to valence-bearing contexts. Value can be read from the felt need-consequences of lived routes. X is therefore not an arbitrary score. It is read from lived structure.

### Consciousness Is Exclusive Process-Occupation
Consciousness is not a passive display of all information in the system. Consciousness is the active occupation of one process-space where continuity must be stabilized.

The agent cannot fully execute, observe, reflect, feel, and inspect surprise at the same time. Each process requires a different commitment. Recognize–Execute commits to a living belief. Try–Observe holds and varies context-skill relations. Reflection suspends execution and searches the belief-horizon. Need-feeling occupies felt pressure. Perceptual surprise occupies the changed Context-surface. The processes can switch rapidly, but one must dominate. Where attention points, consciousness works.

### Agency Begins Socially
The individual agent does not begin life with fully private beliefs. At birth, the agent is carried by shared and provider-held belief-fields. Others regulate Needs, organize contexts, select actions, and maintain routes to relief and fulfillment before the infant can do so alone.

Early learning is therefore the gradual internalization of shared beliefs. What later appears as private agency begins as externally carried continuity. Caregivers and groups hold the routes first; the child gradually differentiates, stabilizes, and executes them from within. Social dependency is not a later addition to agency. It is one of its developmental origins.

### Play and Imagination Are Consequence-Bracketed Processing
Play and imagination use the same processes as obligated cognition, but with reduced consequence-binding. When active Needs are sufficiently covered, the agent can loosen the requirement that every belief be immediately realizable in the external world.

This allows the agent to try routes, rehearse actions, simulate social situations, explore variants, and traverse possible belief-loops without full commitment. Failed play does not necessarily threaten continuation. Failed imagination does not necessarily damage the actual context. This makes play and imagination central to learning: they expand the Belief-horizon while keeping real-world cost low. Under pressure, consciousness is obligated to secure a route. Under surplus, the same machinery becomes free to explore.
